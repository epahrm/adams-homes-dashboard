# Porter Travel Planner

Private family travel app & planning docs for Elizabeth Porter.

- `index.html` — the entire app: static, self-contained, offline-first PWA
  (no build step, no dependencies; deploys on Vercel as-is)
- `sw.js` — service worker; bump `VERSION` with any app change
- `docs/` — planning standards, preferences, city files, trip files,
  and trip journals (durable memory for the EP Travel Planner)

Unlisted/private — shared only by Elizabeth.
